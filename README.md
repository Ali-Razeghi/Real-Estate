# LuxeVista Realty – Real Estate Website

A modern, responsive, and visually refined real estate website designed as a professional portfolio project.
This project demonstrates front-end development skills with a focus on UI/UX design, layout structure, and user interaction.

## Project Overview

LuxeVista Realty is a fully responsive real estate website that simulates a premium property agency platform.
It includes property listings, search functionality, agent profiles, testimonials, and a contact system.

This project is designed to showcase:

* Clean UI/UX design principles
* Responsive layout techniques
* Real-world business website structure

---

## Features

* Modern Hero section with search panel
* Property listings with categories and filters
* Dual action buttons (Request Details / Book Viewing)
* About section with company overview
* Services section
* Featured neighborhoods
* Agent profiles
* Testimonials
* FAQ section
* Contact form (Netlify-ready)
* Smooth scrolling and animations
* Fully responsive design (mobile-friendly)

---

## Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
* Vanilla JavaScript
* Responsive Design Principles

## Installation & Usage

1. Clone the repository:

```
git clone https://github.com/YOUR-USERNAME/real-estate-website.git
```

2. Open the project folder

3. Run the project:

* Simply open `index.html` in your browser

---

## Deployment

This project is optimized for deployment on:

* Netlify
* GitHub Pages

To deploy on Netlify:

1. Upload your project folder
2. Set `index.html` as the entry point
3. Publish
